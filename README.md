# Resepti tietokanta sovellus

- Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen
- Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan reseptejä
- Käyttäjä näkee kaikki lisätyt reseptit
- Käyttäjä pystyy etsimään reseptiä hakusanalla (to do)
- Reseptejä pystyy luokittelemaan esim. alku-, pää-, jälkiruoka, ranskalainen, vegaaninen yms. (to do)
- Sovelluksessa on käyttäjäsivut, joilla näkyy tilastoja ja ilmoituksia käyttäjään liittyen. (to do)
-Sovelluksessa pystyy kommentoimaan muiden käyttäjien reseptejä (to do)
